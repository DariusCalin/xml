Introducción a Lenguaje de Marcas y Sistemas de Gestión de Información (LMSGI)

¿Qué son los LMSGI?

Un lenguaje de marcas es un sistema para anotar un documento de manera que se distingue el contenido del texto de algunas características del texto. Los lenguajes de marcas más conocidos son HTML y XML. Estos lenguajes usan "marcas" o "etiquetas" para definir diferentes elementos dentro de un documento.

Las etiquetas son palabras clave rodeadas por corchetes angulares, por ejemplo, <etiqueta>. Estas etiquetas se usan para marcar el inicio y el final de los elementos del documento.

Historia

La historia del lenguaje de marcas comienza con la invención del SGML (Standard Generalized Markup Language) en los años 1980. SGML es un estándar para definir lenguajes de marcas para documentos.

HTML (HyperText Markup Language) fue desarrollado en los años 1990 como un lenguaje de marcas específico para crear y estructurar páginas web. XML (eXtensible Markup Language) surgió como una simplificación de SGML para facilitar el intercambio de datos estructurados a través de la web.

Sintaxis

La sintaxis de los lenguajes de marcas generalmente incluye:

  • Elementos: Son las unidades básicas del contenido y están definidos por etiquetas. Por ejemplo, en HTML, un elemento de párrafo se define con <p> y </p>.
  • Atributos: Proporcionan información adicional sobre los elementos. Se colocan dentro de la etiqueta de apertura. Por ejemplo, <a href="https://example.com">enlace</a> define un enlace con el atributo href.
  • Comentarios: Son notas o explicaciones que no se muestran en el resultado final. En HTML, los comentarios se escriben como <!-- comentario -->.

Los lenguajes de marcas son utilizados para estructurar, describir y gestionar datos de manera efectiva, y su sintaxis es fundamental para asegurar la correcta interpretación y presentación de la información.

Principales LMSGI

  • HTML: Utilizado para crear páginas web y aplicaciones web. Permite estructurar contenido y crear enlaces entre páginas.
  • XML: Diseñado para almacenar y transportar datos. Es extensible y permite definir etiquetas personalizadas.
  • Markdown: Un lenguaje de marcas ligero para formatear texto de manera sencilla. Muy popular en la documentación de software y blogs.

Aplicaciones y Uso

Su uso en la actualidad se puede presenciar en cualquier lugar, desde paginas webs, bases de datos, etc..
