Introducción a XML

XML (Extensible Markup Language) es un lenguaje de marcado que define un conjunto de reglas para la codificación de documentos en un formato que es legible tanto para humanos como para máquinas.

Historia

XML fue desarrollado por el World Wide Web Consortium (W3C) en 1996. Se basa en SGML (Standard Generalized Markup Language) y se diseñó para ser más simple y más fácil de usar en la web. El objetivo principal de XML es facilitar el intercambio de datos a través de diferentes sistemas y plataformas.

Características de XML

  • Legibilidad: XML es legible tanto por humanos como por máquinas.
  • Extensibilidad: Permite definir etiquetas personalizadas según las necesidades del usuario.
  • Estructura: Los documentos XML tienen una estructura jerárquica clara.
  • Interoperabilidad: Facilita el intercambio de datos entre diferentes sistemas.
